Identifying Customer Segments - Unsupervised Learning

This project is divided into three parts:
1. Preprocessing Data 
a. Converting missing codes to NaN
b. Accessing missing data in rows and columns
c. Re-encode categorical features
d. Engineer Mixed Type features
e. Complete Feature Selection

2. Feature Transformation
a. Apply Feature Scaling
b. Perform Dimensionality Reduction
c. Scree Plots
d. Interpret PCA Components

3. Clustering - Applying KMeans learning
